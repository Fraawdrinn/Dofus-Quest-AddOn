# Dofus-Quest-AddOn

## Add-on overlay for Dofus players

# For developers
## Requirement
### On Windows

Use command: 'DQA-venv\Scripts\activate' to work with python libraries
Then, use command: pip install -r requirement.txt

### On Linux/MacOS

Use command 'source DQA-venv/bin/activate' --
Then, use command: pip install -r requirement.txt
